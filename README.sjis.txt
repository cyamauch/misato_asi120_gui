
■これは?

  紀美野町みさと天文台の星空ツアー(105cm大型望遠鏡使用の観望会)で ASI120 カメラ
を試用するための補助 GUI です．ASI120付属ドライバ・ツールとTcl/Tkとが動く環境
(Windows)で使用する事ができます．Windowsタブレットで使う事を想定しています．

■使い方

 - Tcl/Tk をインストールします．付属の ActiveTcl*-win32-ix86-threaded.exe を実
   行して，インストールしてください．本ツールでは Tcl/Tk の最低限の機能しか必要
   としませんから，インストール中に警告が出てもかまいません．無視して続行してく
   ださい．

 - 本ツールのディレクトリ構造を変更せず，任意のディレクトリに一式コピーしてくだ
   さい．

 - kanbokai_gui.tcl をメモ帳などで開き，最初の exec_file の値を適切なものに変更
   してください．コードのコメントをみて，必要に応じてフォントやAMCapウィンドゥ
   の位置・サイズを変更してください．

 - kanbokai_gui.tcl をダブルクリックすると，実行できます．

 - ダーク引きの ON/OFF については，現状ではうまくいっていません．
